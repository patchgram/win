# Список патчей Patchgram

Здесь простым языком описано, какие патчи есть в Patchgram и что они делают.

## Типы патчей

| Тип | Что это значит | Когда используется |
| --- | --- | --- |
| `dll` | Patchgram инжектит свою runtime-библиотеку `Patchgram.dll` в Telegram Desktop и меняет поведение во время работы клиента. | Визуальные патчи, настраиваемые значения и функции, которые удобнее обновлять без постоянной перезаписи большого количества байтов в клиенте. |
| `binary` | Patchgram меняет найденные byte-паттерны прямо в `Telegram.exe`. | Патчи запросов, constructor id и локальных проверок, которые проще и стабильнее менять напрямую в бинаре. |

> Большинство патчей работают только локально на клиенте. Они меняют то, что Telegram Desktop отправляет, блокирует или показывает у тебя. Они не меняют серверные данные аккаунта Telegram.

## Патчи

Главный экран Patchgram группирует патчи по пяти разделам — **Accounts**, **Messages**, **Optimizations**, **Gifts** и **Misc**. Каждый патч ниже указан в том разделе, в котором он находится.

### Accounts

| Патч | Тип | Что делает |
| --- | --- | --- |
| Always offline | `dll` | Держит аккаунт оффлайн локально, принудительно меняя значение offline в `account.updateStatus`. |
| Don't share phone when adding contacts | `dll` | Не даёт Telegram Desktop отправлять флаг исключения приватности номера при добавлении контакта. |
| 999 accounts | `binary` | Поднимает локальный лимит аккаунтов в Telegram Desktop до 999. |
| Hide self phone | `dll` | Возвращает пустой номер в профиле для твоего собственного аккаунта, скрывая строку с телефоном в своём профиле. |
| Custom account settings | `dll` | Общий патч локальной кастомизации аккаунта: балансы, level rating, бейджи, bot verification, Premium, номер, userID, канал в профиле, Fragment phone, кастомные юзернеймы и заморозка аккаунта. Подпатч заморозки показывает аккаунт замороженным, подставляя даты заморозки и URL апелляции в локальный ответ `help.appConfig`. |

### Messages

| Патч | Тип | Что делает |
| --- | --- | --- |
| Message settings | `dll` | Общий патч приватности для статуса печати, прочтений, локальных черновиков, локального Fact Check под постами каналов, защиты копирования/сохранения и TTL. |
| Show bot callback-data on hover | `binary` | Показывает callback-data inline-кнопок бота при наведении/копировании текста кнопки. |
| Sensitive blur | `binary` | Отключает локальный блюр sensitive-контента. |
| More recent stickers | `binary` | Поднимает лимит отображения недавних стикеров с дефолтных 20 до 200 — в ряду «Recent» в панели стикеров показывается намного больше недавно использованных. Runtime-патч памяти по `kRecentDisplayLimit` в `StickersListWidget::collectRecentStickers`. |
| Open links without warning | `binary` | Открывает скрытые/внешние ссылки без дополнительного предупреждения Telegram. |
| Disable media spoilers | `binary` | Показывает фото/видео со спойлером сразу, без локального блюра. |

### Optimizations

| Патч | Тип | Что делает |
| --- | --- | --- |
| Disable Premium, Stars, TON & Gifts | `dll` | Отключает выбранные monetization-разделы и запросы через runtime: Premium, Stars, TON, Gifts, boosts, paid reactions, emoji statuses и части app config. |
| Disable premium effects | `binary` | Отключает локальный запуск Premium-эффектов и анимаций стикеров. |
| Hide stories | `dll` | Скрывает stories локально и блокирует известные запросы получения/прочтения/просмотра stories. |
| Disable ads | `dll` | Отключает Telegram Ads и промо от proxy sponsor. |

### Gifts

Все патчи Gifts — `dll`: они работают в runtime, переписывая ответы `payments.savedStarGifts` / `payments.starGifts` внутри `Patchgram.dll`. Каждая перезапись валидируется и откатывается при ошибке, чтобы не дестабилизировать клиент. Байты Telegram не патчатся.

| Патч | Тип | Что делает |
| --- | --- | --- |
| Spoof profile gifts | `dll` | Локально подменяет звёздные подарки, показываемые в профиле, переписывая ответ `payments.savedStarGifts` в runtime-библиотеке. У патча своё окно настроек: задаёшь отправителя (user/channel/chat, id в стиле Bot API), дату, id подарка, цену в Stars, значение конвертации в Stars, подпись, саплай (доступно/всего) и значки (Limited, Can upgrade с ценой апгрейда, Auction с названием и номером подарка, Was refunded). Также может заменить стикер подарка на кастом-эмодзи: впиши id кастом-эмодзи или нажми **Get id from gift** (ищет его в `api.changes.tg`), затем один раз открой пак этого эмодзи, чтобы Patchgram захватил его полный документ и подставил (используй анимированный TGS/WEBM-эмодзи, чтобы он отображался внутри подарка, а не только в списке). «Save & Apply» применяет на лету к запущенному Telegram — переоткрой профиль, чтобы обновить. Байты Telegram не патчатся. |
| Spoof profile unique gifts | `dll` | Локально показывает подарок в профиле как улучшенный (уникальный), переписывая ответ `payments.savedStarGifts` в runtime-библиотеке. Своё окно настроек: выбираешь улучшаемый подарок (каталог из `api.changes.tg`) или **Empty** для полностью кастомных id, затем задаёшь название, уникальный номер, модель, символ, фон (backdrop), счётчики выпущено/всего, value и last-resale, и идентичность (отправитель, владелец, host, owner address, дата). Работает с уже уникальными подарками и конвертирует обычные в уникальные. Для сконвертированного подарка также локально отвечает на запрос деталей стоимости, чтобы они показывались, а не падали с ошибкой. «Save & Apply» применяет на лету — переоткрой профиль; выбор чьего профиля (только я / все / все кроме меня). Байты Telegram не патчатся. |
| Fake transfer | `dll` | Локально делает подменённый подарок передаваемым и фейкает передачу. Добавляет кнопку **Transfer** к подарку в профиле и в окне отправки; при передаче в чате с получателем появляется сервисное сообщение о передаче. С включённым патчем даже реальный подарок на самом деле не передаётся — запрос делается невалидным, поэтому на сервер ничего не уходит и получатель ничего не получает; сообщение только локальное и пропадает после перезапуска. Требует включённого **Spoof profile unique gifts**. Байты Telegram не патчатся. |
| Show hidden gifts | `dll` | Локально добавляет дополнительные звёздные подарки в меню покупки, дописывая записи в ответ `payments.starGifts` в runtime-библиотеке. Внедрённые подарки (цена 50 Stars, не лимитированные) показываются в **начале** списка; их стикеры используют соответствующий кастом-эмодзи, найденный в `api.changes.tg`. Байты Telegram не патчатся. |

### Misc

| Патч | Тип | Что делает |
| --- | --- | --- |
| DLL injection | `dll` | Инжектит runtime-библиотеку Patchgram (`Patchgram.dll`) в Telegram Desktop через лаунчер. Настоящий `Telegram.exe` переименовывается в `Telegram_real.exe`, а его место занимает небольшой лаунчер; при каждом запуске лаунчер стартует настоящий клиент в приостановленном состоянии, инжектит `Patchgram.dll`, затем возобновляет его — так каждый запуск оказывается пропатченным. Базовый хук, через который грузятся все runtime-патчи; сам по себе просто загружает библиотеку без изменения поведения. |
| MTProto request/response logger | `dll` | Логирует каждый MTProto-запрос, который отправляет Telegram, и каждый ответ — каждый с отметкой времени — в `PatchgramHook.log` рядом с `Telegram.exe`. Логгер открывает свой файл в конструкторе внедрённой библиотеки, до того как Telegram отправит первый пакет, поэтому трасса полная с самого запуска. Каждая строка — **полностью декоднутый TL**: метод/тип и его рекурсивно разобранные поля (flags, векторы, вложенные объекты, строки) встроенным TL-декодером, так что лог читаем без внешних инструментов. При включении автоматически включается DLL injection. Рисуется `Patchgram.dll` — байты Telegram не патчатся. |

## Подпатчи Message Settings

| Подпатч | Что делает |
| --- | --- |
| Typing activity | Не отправляет индикатор "печатает". |
| Read receipts | Не отправляет запросы прочтения через пропатченные пути. |
| Local drafts | Оставляет черновики локальными, блокируя синхронизацию drafts. |
| Custom Fact Check | Локально запускает в Telegram Desktop путь запроса Fact Check для видимых постов и подменяет ответы `messages.getFactCheck` на твой текст. |
| Copy/save protect content | Локально принудительно сбрасывает в false флаг `noforwards` (flags.26) у `message#7600b9d3`, чтобы можно было копировать текст и сохранять медиа из чатов/каналов с запретом сохранения. (Пересылку всё ещё может блокировать отдельное ограничение на уровне канала.) |
| Disable TTL | Локально отключает таймеры самоуничтожения/автоудаления: принудительно ставит `ttl_seconds` медиа (одноразовые видео/документы) в 0 при создании и обнуляет время автоудаления сообщения `ttl_period`, чтобы временные сообщения не удалялись локально. |

## Подпатчи Disable Premium, Stars, TON & Gifts

| Подпатч | Что делает |
| --- | --- |
| App config | Блокирует запрос monetization app config. |
| Premium UI | Локально скрывает или отключает Premium-разделы в интерфейсе. |
| Gifts | Локально скрывает или блокирует действия, связанные с подарками. |
| Paid reactions | Локально блокирует paid reactions: доступность, отправку и обработку. |
| Emoji statuses and effects | Локально скрывает или блокирует emoji statuses и связанные Premium-эффекты. |
| Stars, TON and collectibles | Локально скрывает или блокирует Stars, TON и collectible-разделы монетизации. |
| Boosts | Локально скрывает или отключает boost-действия. |
| Read receipts fix | Сохраняет корректное поведение меню "кто прочитал" вместе с monetization-патчем. |

## Подпатчи Custom Account Settings

| Подпатч | Что делает |
| --- | --- |
| Custom Stars | Визуально меняет баланс Stars в My Stars и пунктах монетизации. |
| Custom TON | Визуально меняет баланс TON в My TON и пунктах монетизации. |
| Custom level rating | Визуально меняет level/rating Stars для выбранных пользователей. |
| Visual peer badge | Визуально добавляет локальный бейдж Verified, Scam или Fake выбранным юзерам/каналам. |
| Bot verification | Визуально добавляет локальную bot verification. Можно выбрать, где показывать и какой preset использовать. |
| Local Telegram Premium | Локально включает Premium-доступность для UI-проверок Telegram Desktop. |
| Account freeze | Показывает аккаунт замороженным, подставляя даты заморозки и URL апелляции в локальный ответ `help.appConfig`. |
| Custom phone number | Визуально заменяет твой номер телефона локально. Пустая строка означает оригинальный номер. |
| Custom userID | Визуально заменяет отображаемый userID локально. Пустая строка означает оригинальный ID. |
| Local attached channel | Визуально прикрепляет другой канал по channel ID. Чтобы он корректно отображался у тебя в клиенте, сначала нужно открыть/прогрузить этот канал в Telegram Desktop. |
| Fragment phone | Локально делает отображаемый номер коллекционным и позволяет задать свои значения `fragment.collectibleInfo`. |
| Custom list usernames | Локально заменяет список юзернеймов в твоём профиле. Юзернейм можно сделать обычным или collectible, а для collectible задать свои значения `fragment.collectibleInfo`. |
